The following are included in this zip file: accessCOMPASS.htm and accessCOMPASS.aspx

This application (accessCOMPASS.htm) is designed to be placed on your 
secured intranet server, at the location you've specified.

Users who wish to enter the COMPASS site must first visit this
intranet URL to access the subscription.


For SharePoint Users: 

The accessCOMPASS.aspx is an active server page for customers with SharePoint.
In this case put both files in a SharePoint folder at the location you've specified.


If you have access issues, or problems connecting, please make
sure that the file has been placed in the specified location. To make changes 
to the location or to reach our technical support, please email us at support@astm.org.
